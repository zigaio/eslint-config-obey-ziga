# no-extra-parens

## Details

```javascript
'no-extra-parens': [2, 'all', {
  nestedBinaryExpressions: false,
  ignoreJSX: 'all',
}]
```

## Reason

Stylistic preference.

I prefer wrapping JSX in parentheses as a quick visual indication of where the JSX declarations begin and end.
